# 📅 Interactive Calendar Component

An interactive, responsive calendar component built with Next.js and Tailwind CSS. Inspired by a physical wall calendar layout, this project combines functionality with a clean modern UI.

## ✨ Features

- 📅 Date range selection (start & end)
- 🎯 Visual highlighting of selected range
- 📝 Notes section for user input
- 💾 Data persistence using localStorage
- 📱 Fully responsive design (mobile + desktop)
- 🎨 Modern UI with gradient styling

---

## 🛠 Tech Stack

- Next.js
- React
- Tailwind CSS
- date-fns

---

## 🧠 Key Design Decisions

- **Component-based architecture** for scalability  
  (CalendarGrid, DayCell, NotesPanel)
  
- **Tailwind CSS** for rapid and responsive styling  

- **LocalStorage** used instead of backend (as per requirements)

- Focus on **UX clarity** with visual feedback for date selection
